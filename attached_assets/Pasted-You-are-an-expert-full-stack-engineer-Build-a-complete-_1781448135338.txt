### GOAL
Create a browser-based IDE with:
- Monaco Editor (VS Code editor)
- Multi-file explorer
- Terminal emulator
- Backend execution engine (run Python, Node, Bash)
- Virtual filesystem (JSON or real FS)
- AI prompt box for code generation, refactoring, debugging
- Live output panel
- Ability to run code in subprocesses
- WebSocket or HTTP streaming for output

       - Fix errors
       - Explain code
       - Refactor code

5. **File System**
   - Use real Replit filesystem OR
