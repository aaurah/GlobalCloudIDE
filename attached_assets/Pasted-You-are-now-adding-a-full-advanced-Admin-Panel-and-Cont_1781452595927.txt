### NEW FEATURES TO ADD

1. **Admin Roles & Permissions**
   - Add roles:
       - super_admin
       - admin
       - support
       - auditor
   - Restrict access to admin endpoints and UI.
   - Endpoints:
       - /admin/roles
       - /admin/users/assign-role

2. **User & Team Management**
   - View:
       - users
       - teams
       - roles
       - usage
   - Actions:
       - suspend user
       - reset password (or send reset link)
       - change role
       - view team members
   - Endpoints:
       - /admin/users/list
       - /admin/users/{id}
       - /admin/users/suspend
       - /admin/teams/list

3. **Project & Deployment Oversight**
   - View:
       - all projects
       - all deployments
       - status
       - regions
       - resource usage
   - Actions:
       - force stop deployment
       - restart deployment
       - lock project
   - Endpoints:
       - /admin/projects/list
       - /admin/deployments/list
       - /admin/deployments/stop
       - /admin/projects/lock

4. **Node & Region Control**
   - View:
       - nodes
       - regions
       - capacity
       - health
   - Actions:
       - drain node
       - disable region
       - rebalance workloads
   - Endpoints:
       - /admin/nodes/list
       - /admin/nodes/drain
       - /admin/regions/disable

5. **Billing & Credits Admin**
   - View:
       - user credits
       - usage history
       - invoices (if any)
   - Actions:
       - adjust credits
       - grant promo credits
   - Endpoints:
       - /admin/billing/users
       - /admin/billing/adjust

6. **Agents & Plugins Governance**
   - View:
       - installed agents
       - installed plugins
       - usage
   - Actions:
       - approve/reject plugins
       - disable agent for a user or team
   - Endpoints:
       - /admin/agents/list
       - /admin/agents/disable
       - /admin/plugins/list
       - /admin/plugins/approve

7. **Security & Guardrails**
   - Configure:
       - rate limits
       - max resources per user
       - allowed regions
       - AI safety/guardrail settings
   - Endpoints:
       - /admin/security/config
       - /admin/security/limits

8. **Logs, Audits & Observability**
   - View:
       - system logs
       - audit logs (who did what, when)
       - error logs
   - Filters:
       - by user
       - by project

   - Endpoints:
       - /admin/logs
       - /admin/audit
