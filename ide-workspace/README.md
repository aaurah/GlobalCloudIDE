# CloudIDE Workspace

Welcome to your cloud IDE! This is your personal workspace.

## Getting Started

- Browse files in the **File Explorer** on the left
- Click any file to open it in the **Monaco Editor**
- Press **Ctrl+S** to save changes
- Click **▶ Run** to execute your code
- Use the **AI Assistant** panel on the right for Generate, Fix, Explain, or Refactor

- `projects/hello-python/` — Python starter
